# Panduan Belajar Web Development (Kurikulum W3Schools)

Materi ini disusun berdasarkan alur belajar standar **W3Schools.com**.
Setiap file adalah satu bab materi. Buka kodenya untuk membaca penjelasan detail di dalamnya.

## Urutan Belajar

### 1. HTML (Pondasi)
Buka folder `/html`:
- `01-intro-basic.html`: Mulai dari sini. Pengenalan tag dasar.
- `02-formatting.html`: Mengatur teks (Tebal, Miring, Judul).
- `03-links-images.html`: Cara pasang Link dan Gambar.
- `04-list-table.html`: Membuat Daftar dan Tabel data.
- `05-block-inline.html`: Memahami elemen kotak (Div) vs baris (Span).
- `06-forms.html`: Membuat formulir input data.
- `07-media.html`: Pasang Video & Audio.
- `08-semantic.html`: Struktur website yang benar (Header, Footer, dll).

### 2. CSS (Desain)
Buka folder `/css`:
- `01-syntax-selectors.css`: Cara memilih elemen yang mau dihias.
- `02-colors-backgrounds.css`: Warna dan gambar latar.
- `03-box-model.css`: PADDING, MARGIN, BORDER (Wajib Paham!).
- `04-text-fonts.css`: Mengatur tulisan.
- `05-layout-position.css`: Mengatur posisi elemen (Absolute, Fixed, Sticky).
- `06-flexbox.css`: Layout modern 1 dimensi.
- `07-grid.css`: Layout modern 2 dimensi.
- `08-responsive.css`: Membuat web bagus di HP (Media Queries).

### 3. JavaScript (Logika)
Buka folder `/js`:
- `01-syntax-variables.js`: Dasar variabel dan operator.
- `02-data-types.js`: Tipe data (String, Angka, Object).
- `03-strings-numbers.js`: Manipulasi teks dan hitungan.
- `04-arrays.js`: Mengolah daftar data (Penting!).
- `05-logic-loops.js`: Logika IF/ELSE dan Perulangan.
- `06-dom.js`: Cara mengubah HTML lewat JS (Interaktif).
- `07-es6-modern.js`: Fitur JS Modern (Arrow Function, Async/Await).
