1. HTML Canvas API

Canvas adalah area gambar yang bisa dikontrol dengan JavaScript.

2. SVG (Scalable Vector Graphics)

SVG adalah grafik berbasis XML (bisa di-zoom tanpa blur).

Perbedaan Canvas vs SVG

  • Canvas:
    • Berbasis Pixel (Raster).
    • Digambar dengan Script (JS).
    • Performa tinggi untuk banyak objek bergerak (Game).
    • Tidak bisa mendeteksi event click pada objek spesifik (harus hitung koordinat).
    • Save as PNG/JPG mudah.
  • SVG:
    • Berbasis Vektor (Shape).
    • Dibuat dengan Tag XML.
    • Tetap tajam di resolusi berapapun (Retina display).
    • Setiap shape adalah elemen DOM (bisa kasih class/id/onclick).
    • Kurang bagus untuk game berat.
