HTML File Paths

    CONTOH RELATIVE PATH:
    
    <img src="picture.jpg">           ← File di folder yang SAMA
    <img src="images/picture.jpg">    ← File di folder 'images' (masuk ke dalam)
    <img src="/images/picture.jpg">   ← File di ROOT folder (paling atas)
    <img src="../picture.jpg">        ← File di folder ATAS (naik 1 tingkat)
