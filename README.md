## ğŸ“ Cara Menggunakan

### Untuk Materi Teori

1. Buka folder `/html`, `/css`, atau `/js`
2. Mulai dari file `01-...`
3. Baca file di Text Editor (VS Code recommended)
4. Fokus baca komentar di setiap baris kode
5. Coba ubah kode dan lihat hasilnya

### Untuk Live Projects

1. Buka folder `/projects`
2. Pilih project sesuai level kemampuan
3. Buka file `.html` di browser
4. Inspeksi kode (View Source) untuk belajar
5. Modifikasi dan eksperimen!
